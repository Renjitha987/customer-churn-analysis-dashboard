# Customer Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn data for a telecommunications company using Power BI. The dashboard helps identify customer behavior, churn trends, and business insights to improve customer retention.

## Features
- KPI Overview
- Churn Rate Analysis
- Customer Demographics
- Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method
- Customer Tenure Analysis
- Interactive Filters/Slicers
- Business Insights Section

## Tools Used
- Power BI
- DAX
- Data Visualization
- Data Cleaning
- Dashboard Design

## Key Insights
- Month-to-month contracts show higher churn.
- Fiber optic customers churn more frequently.
- Electronic check users have higher churn.
- Long-term customers are more loyal.
